Capturing and Postprocessing software

Capturing Software

Iris to capture planetary video. It is freeware.

PHD for autoguiding. It is freeware.

Canon EOS Utility to capture deep sky data. It is provided on any Canon DSLR. Canon EOS allows to focus precisely (provided a bright enought star), configure all camera parameters and tethered shooting. If you have autofocus lens, you can all that remotely (with Windows Terminal Desktop or similar). I hope in the future implement some sort of LiveView exposure control to allow to focus on dimmer objects (stars).

Cartes du Ciel to track telescope position through ASCOM hub. I prefer this to Stellarium in the telescope scope PC since Stellarium is not Desktop Terminal friendly.

Postprocessing Software

Registax for plantery video stacking and wavelet processing. This is freeware.

DeepSkyStacker for deep sky photography stacking. This is freeware.

Adobe Photoshop for final image processing. It is payware.

Stellarium to compare pictures and preparing composition.
